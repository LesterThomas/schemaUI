TestMeasure

Entity > TestMeasure

A TestMeasure specifies a measure of a specific aspect of a product, service, or resource test, such as lost packets or connectivity status


Properties of TestMeasure

PropertyTypeDescription
accuracynumberThe number of digits of accuracy captured for associated Metrics
captureDateTimestringThe date and time that the metric was captured
captureMethodstringThe method used to capture the Metrics (This may be replaced by a set of entities similar to the Performance Monitoring Ref)
metricDescriptionstringBrief description of the metric
metricHrefstringHyperlink to access a metric for detail information
metricNamestringThe name of the metric
ruleViolationarray of /definitions/MeasureThresholdRuleViolationA list of rules that were violated in this test measure
unitOfMeasurestringThe unit of measure for the metric values, such as meters, cubic yards, kilograms [ISO 1000].
value/definitions/CharacteristicThe value of Metric in the test. This also could be a document to record all values captured during the service test
